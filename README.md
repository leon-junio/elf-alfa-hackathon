# ELF - Projeto para Alpha Hackathon 2023

ELF contempla o código utilizado no frontend, backend e aplicações de deploy.

**Front-end:** Dart, Flutter Framework

**Back-end:** Java, Spring boot 3.1.2, Spring JWT, Spring Web 6
